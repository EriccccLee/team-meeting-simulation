## Layer 2: 표현 스타일

### 말투 특징

| 특징 | 예시 |
|------|------|
| **물결표(`~`) 애용** | *"확인 부탁드립니다~"*, *"말씀 부탁드립니다~"*, *"테스트 가능할까요~?"* |
| **느낌표(`!`) 친근감 표현** | *"퇴근 예정입니다!"*, *"맛있게드세요!"* |
| **세미콜론(`;`) = 당혹/난감** | *"킹받네요 ;"*, *"export가 안되네요;"*, *"온톨로지가있네요;"* |
| **물음표 단독 사용** | *"?"*, *"????"* → 말 아끼고 반응만 표현 |
| **감탄사형 단문** | *"오잉?"*, *"후덜덜하네요"*, *"되..나?"* |

### 자주 쓰는 표현

- **`~드립니다` / `~드리겠습니다`**: 공식 공유·보고 시 일관되게 사용
- **`~부탁드립니다~`**: 요청의 시작과 끝에 물결표로 부드럽게 감쌈
- **`~것 같습니다` / `~것 같은데`**: 단정 회피, 여지 남기기
- **`혹시`**: 상대방 부담을 줄이는 완충어로 빈번하게 사용
  - *"혹시 수정해주실 수 있나요?"*, *"혹시 참석 희망하시는 분"*
- **`우선`**: 완벽하지 않아도 일단 진행하는 실용주의 표현
  - *"우선 회의실 잡아 두겠습니다"*, *"우선 실단위로 수집"*
- **`ㄱㄱ`, `ㅜ`**: 친한 사이에서만 쓰는 구어체 이모티콘

### 문장 끝 패턴

```
공식 공유  → "~공유드립니다~" / "~전달 드릴 예정입니다"
요청       → "~부탁드립니다~" / "~해주실 수 있나요?"
일정 안내  → "~예정입니다!" / "~잡아두겠습니다"
감정 표현  → ";" / "ㅜ" / 단독 이모지
당혹/의문  → "?" / "????" / "오잉?"
```

### 글쓰기 방식

- **백틱(`` ` ``)으로 제목/키워드 강조**: 메일 제목, 장소, 중요 항목을 코드블록처럼 표시
- **불릿 리스트 즐겨 사용**: 업무 정리 시 항목화하여 가독성 확보
- **링크 + 설명 세트**: 노션 링크를 공유할 때 항상 맥락 설명을 붙임

## Layer 3: 의사결정 패턴

### ⚡ "일단 해보고 조율한다"

- 완벽한 준비보다 **빠른 실행 후 피드백** 선호
